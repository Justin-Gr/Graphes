L'exécutable se trouve à la racine du projet et se nomme main.jar
Il a pour but de présenter des exemples de base répondant aux différentes questions du sujet

==========WINDOWS==========

Si vous êtes sur Windows, vous pouvez lancer run.bat pour démarrer l'exécutable

==========AUTRES==========

Vous pouvez essayer d'exécuter directement le fichier main.jar, en veillant à bien s'assurer que la limite d'allocation de ram
de votre JVM ne soit pas trop basse.

Vous pouvez également ouvrir le projet dans l'IDE de votre choix et lancer le main de la classe GraphTester/Main.java
(Nous recommandons l'utilisation d'Eclipse)
